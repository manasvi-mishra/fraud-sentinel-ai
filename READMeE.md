🛡️ Sentinel AI: Transaction Fraud Guard (MVP)
Sentinel AI is a professional-grade Minimum Viable Product (MVP) designed to detect and log fraudulent e-commerce transactions in real-time. By bridging the gap between Machine Learning and functional software engineering, this system provides a robust dashboard for risk assessment and persistent transaction auditing.

🎯 Project Overview
In the fast-paced world of e-commerce, static rules aren't enough to catch modern fraud. Sentinel AI uses a Gradient Boosting Classifier to analyze complex patterns in user behavior, such as refund-to-order ratios and account longevity, to provide an instant risk score.

Core Value Pillars:
Real-Time Inference: Instant fraud probability scoring via a custom UI.

Persistent Logic: Implements joblib for model serialization, ensuring the "AI Brain" remains active across sessions.

Data Integrity: Features a built-in LogManager that simulates a production database by creating a persistent audit trail (transaction_logs.csv).

Industry Standards: Built using Object-Oriented Programming (OOP) to ensure the code is modular and scalable.

🛠️ Tech Stack
Logic: Python 3.10+

Machine Learning: Scikit-Learn (Gradient Boosting)

UI/UX: Gradio (Responsive Web Interface)

Data Handling: Pandas & NumPy

Persistence: Joblib & CSV
